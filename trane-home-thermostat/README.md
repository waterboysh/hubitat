Install Instructions:

1. In your Hubitat menu, go to Apps Code
2. Click the "+ Add app" button
3. Click the 3 dot menu in the top right and choose Import
4. Paste the following URL: https://raw.githubusercontent.com/waterboysh/hubitat/main/trane-home-thermostat/nexiaThermostatManager.groovy and click Import
5. Click Yes to confirm
6. Click the save button in the top right
7. From the Hubitat menu, go to Drivers Code
8. Repeat steps 2 - 6, but paste the following URL in step 4: https://raw.githubusercontent.com/waterboysh/hubitat/main/trane-home-thermostat/nexiaThermostat.groovy
9. In the Hubitat menu, go to Integrations
10. Click the "+ Add user integration" button
11. Choose Nexia Thermostat Manager
